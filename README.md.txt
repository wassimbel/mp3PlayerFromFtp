# My Project

## Overview
This project is a C++ application designed to run on an embedded Linux platform, such as a Raspberry Pi. The application connects to an FTP server, downloads MP3 files, plays them sequentially using VLC, and sends playback information over a UART connection.

## Requirements
- CMake
- libcurl
- VLC
- Google Test

## Setup Instructions

### Installing Dependencies

#### libcurl
sudo apt-get update
sudo apt-get install libcurl4-openssl-dev
sudo apt-get install vlc
